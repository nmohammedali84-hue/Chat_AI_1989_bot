# Multi AI Debate Bot 🤖

بوت تيليغرام يجمع أذكى نماذج الذكاء الاصطناعي في نقاش واحد.

## النشر على Railway

1. روحي على railway.app وسجلي بحساب GitHub
2. اضغطي "New Project" ثم "Deploy from GitHub repo"
3. ارفعي الملفات على GitHub أولاً (أو استخدمي Railway CLI)
4. في إعدادات المشروع، أضيفي متغيرات البيئة:
   - TELEGRAM_TOKEN = توكن البوت من BotFather
   - ANTHROPIC_API_KEY = مفتاح Anthropic API

## الأوامر

- /start — بداية
- /models — اختاري النماذج
- /rounds — عدد الجولات
- /ask — ابدأي النقاش
- /status — إعداداتك الحالية
